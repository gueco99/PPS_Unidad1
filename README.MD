Conversión Binaria, Utilidades de Lista y Pruebas Unitarias
Autor: Diego Pacheco

Descripción del proyecto

Este proyecto contiene tres tareas principales desarrolladas en Python:

Tarea 2 – Conversión Binario → Decimal
Implementación de la función esBinario, que verifica si una cadena contiene únicamente 0 y 1.
El programa principal pide un número binario al usuario y lo convierte a decimal.

Tarea 3 – Utilidades de Lista
Implementación de las funciones:

estaEnRango(valor, minimo, maximo) → Comprueba si un valor está dentro de un rango.

estaEnLista(valor, lista) → Comprueba si un valor está dentro de una lista.

Tarea 4 y Tarea 5 – Pruebas Unitarias (unittest y pytest)
Se desarrollan dos baterías de tests para asegurar el correcto funcionamiento de todas las funciones anteriores:

tests_unittest/ usando unittest

tests_pytest/ usando pytest
